# 华科（HUST）博士论文模板 — 标题规则

## Markdown 标题与 Word 样式对照

| Markdown        | Word 样式     | 说明     |
| --------------- | ------------- | -------- |
| `# 一级标题`    | 1.            | 自动编号 |
| `## 二级标题`   | 1.1 / 1.2     | 自动编号 |
| `### 三级标题`  | 1.1.1 / 1.1.2 | 自动编号 |
| `#### 四级标题` | 1.            | 自动编号 |

## 要点

- 所有标题编号由模板自动生成，**不需要手动写入编号**
- 标题级别严格按照 `#` 数量对应，不要跳级

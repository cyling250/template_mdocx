# 华科（HUST）博士论文模板 — 局部规则

## 标题映射

所有标题编号由模板自动生成，**不需要手动写入编号**。

| Markdown | Word 样式 | 说明               |
| -------- | --------- | ------------------ |
| `#`      | Heading 1 | 章（自动编号）     |
| `##`     | Heading 2 | 节（自动编号）     |
| `###`    | Heading 3 | 小节（自动编号）   |
| `####`   | Heading 4 | 次小节（自动编号） |

## 禁止使用 `- ` 列表语法

模板不支持 Markdown `- ` / `* ` 无序列表。以下写法**不允许**：

```
- 横轴间距：6.3m，纵轴间距：5.4m
- 楼面活荷载：6.8 kN/m²
- 永久荷载：20mm水泥砂浆面层
```

如需罗列多项内容，请选择以下替代方案：

**(1) 手动编号（推荐）：**

```
(1) 横轴间距：6.3m，纵轴间距：5.4m
(2) 楼面活荷载：6.8 kN/m²
(3) 永久荷载：20mm水泥砂浆面层
```

**(2) 直接换行：**

```
横轴间距：6.3m，纵轴间距：5.4m
楼面活荷载：6.8 kN/m²
永久荷载：20mm水泥砂浆面层
```

## 换行与段落分隔

### ❌ 禁止

**禁止相邻行连续书写** — Markdown 中连续的多行内容（行与行之间没有空行）会被解析为 Word 中的软换行（Shift+Enter `↓`），而非独立段落。

```
这是第一项
这是第二项
这是第三项
```

上述写法在 Word 中会变成：

```
这是第一项↓
这是第二项↓
这是第三项
```

这种断行破坏段落样式一致性，且无法正确应用段落间距。

### ✅ 正确做法

需要换行或罗列多项内容时，**每项之间必须空一行**，使每个条目成为独立段落：

```
这是第一项。

这是第二项。

这是第三项。
```

如果需要手动编号罗列，同样每项之间空一行：

```
(1) 横轴间距：6.3m，纵轴间距：5.4m。

(2) 楼面活荷载：6.8 kN/m²。

(3) 永久荷载：20mm水泥砂浆面层。
```

## 禁止随意给公式加粗

### ❌ 禁止

在 Markdown 源文件中**禁止对公式（包括行内公式 `$...$` 和块公式 `$$...$$`）添加 Markdown 粗体标记 `**...**`**：

```
**$E = mc^2$**           ❌ 行内公式被粗体包裹
**$$公式内容$$**         ❌ 块公式被粗体包裹
```

### 原因

公式本身已有字体设置（由 Word 模板中的公式样式控制），外层包裹 `**...**` 不产生视觉加粗效果，反而可能干扰模板的公式解析，导致生成的 Word 文档中公式样式异常。

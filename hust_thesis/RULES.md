# 华科（HUST）博士论文模板 — 局部规则

## 标题映射

所有标题编号由模板自动生成，**不需要手动写入编号**。

| Markdown | Word 样式 | 说明 |
|----------|-----------|------|
| `#` | Heading 1 | 章（自动编号） |
| `##` | Heading 2 | 节（自动编号） |
| `###` | Heading 3 | 小节（自动编号） |
| `####` | Heading 4 | 次小节（自动编号） |

## 禁止使用 `- ` 列表语法

模板不支持 Markdown `- ` / `* ` 无序列表。以下写法**不允许**：

```
- 横轴间距：6.3m，纵轴间距：5.4m
- 楼面活荷载：6.8 kN/m²
- 永久荷载：20mm水泥砂浆面层
```

如需罗列多项内容，请选择以下替代方案：

**(1) 手动编号（推荐）：**
```
(1) 横轴间距：6.3m，纵轴间距：5.4m
(2) 楼面活荷载：6.8 kN/m²
(3) 永久荷载：20mm水泥砂浆面层
```

**(2) 直接换行：**
```
横轴间距：6.3m，纵轴间距：5.4m
楼面活荷载：6.8 kN/m²
永久荷载：20mm水泥砂浆面层
```

# 华科（HUST）博士论文模板 — 局部规则

> 本文件为 HUST 模板专属规则，通用语法规则见全局 [RULES.md](../RULES.md)。

## 标题映射

所有标题编号由模板自动生成，**禁止手动写入编号**。

| Markdown | Word 样式 | 说明 |
|----------|-----------|------|
| `#` | Heading 1 | 章 |
| `##` | Heading 2 | 节 |
| `###` | Heading 3 | 小节 |
| `####` | Heading 4 | 次小节 |

### ❌ 禁止

```
# 第1章 绪论
## 1.1 研究背景
```

### ✅ 正确

```
# 绪论
## 研究背景
```

---

## 禁止使用 `- ` 无序列表

HUST 模板不支持 Markdown `- ` / `* ` 无序列表语法。

### ❌ 禁止

```
- 横轴间距：6.3m
- 楼面活荷载：6.8 kN/m²
- 永久荷载：20mm水泥砂浆
```

### ✅ 替代方案一：手动编号（推荐）

```
(1) 横轴间距：6.3m。

(2) 楼面活荷载：6.8 kN/m²。

(3) 永久荷载：20mm水泥砂浆。
```

### ✅ 替代方案二：段落罗列

```
横轴间距：6.3m。

楼面活荷载：6.8 kN/m²。

永久荷载：20mm水泥砂浆。
```

> 无论哪种方案，每项之间必须空一行。详见全局规则「换行与段落分隔」。

# 全局规则 — 扩展 Markdown 语法参考

## 行内格式

| 格式     | 写法       |
| -------- | ---------- |
| **粗体** | `**粗体**` |
| _斜体_   | `*斜体*`   |
| 行内公式 | `$E=mc^2$` |

## 标签前缀约定

| 前缀   | 用途 | 示例标签                    |
| ------ | ---- | --------------------------- |
| `fig:` | 图片 | `@figure[fig:架构图]{描述}` |
| `tbl:` | 表格 | `@table[tbl:对比]{表名}`    |
| `eq:`  | 公式 | `@formula[eq:损失函数]`     |

## 四大规则

1. 每个图/表必须有 `@figure[标签]{描述}` / `@table[标签]{表名}` 声明紧接在前
2. 每个块公式必须有 `@formula[标签]` 声明紧接在 `$$...$$` 之前
3. `@ref{标签}` 的标签必须与声明的标签完全一致
4. `@cite{key}` 的 key 必须存在于 `.bib` 文件中

## 图片

### ✅ 正确写法

```
@figure[fig:架构图]{本文提出的端到端模型架构}
![04-01.png](assets/04-01.png)
```

- `@figure` 紧接在图片行之前
- `[fig:xxx]` 标签前缀必须为 `fig:`

## 表格

### ✅ 正确写法

```
@table[tbl:对比]{不同方法性能对比}
| 方法 | 准确率 |
| --- | --- |
| ResNet | 76.1% |
| ViT | 77.9% |
```

### ❌ 禁止

- 表格中单元格内出现 `$$...$$` 块公式 → 表格内公式一律使用 `$...$` 行内公式
- 表格缺少 `@table[...]{...}` 声明

## 公式

公式分为 **行内公式** 和 **块公式** 两类，规则不同。

### 行内公式书写规则

行内公式指嵌入在文本段落中的公式，使用 `$...$` 包裹。

#### ✅ 正确写法

- 段落中的公式：`损失函数为 $L = L_{cls} + L_{reg}$`
- 表格中的公式：`| $f_c=14.3\text{N/mm}^2$ | 0.003 |`
- 物理单位包裹在 `$...$` 内：`$f_c=14.3\text{N/mm}^2$`

#### ❌ 禁止

- ❌ **禁止单独成行的公式使用 `$...$`** — 单独一行的公式必须用 `$$...$$` 块公式
- ❌ **禁止表格中出现 `$$...$$`** — 表格内的公式使用 `$...$`
- ❌ **禁止单位裸露在 `$...$` 外** — 如 `$f_c=14.3$N/mm²` 应写成 `$f_c=14.3\text{N/mm}^2$`

### 块公式书写规则

块公式指独立占一行的公式，使用 `@formula[eq:标签]` + `$$...$$`。

#### ✅ 正确写法

```
@formula[eq:accuracy]
$$
\text{Accuracy} = \frac{TP + TN}{TP + TN + FP + FN}
$$
```

```
@formula[eq:loss]
$$
\mathcal{L} = \mathcal{L}_{\text{cls}} + \lambda \mathcal{L}_{\text{reg}}
$$
```

#### ❌ 禁止

- ❌ **禁止省略 `@formula[标签]` 声明** — 每个 `$$...$$` 块公式前必须有 `@formula[eq:xxx]`
- ❌ **禁止使用 Mathtype 无法识别的 LaTeX 命令**，包括但不限于：

  | 禁止命令                              | 说明       | 改用                                               |
  | ------------------------------------- | ---------- | -------------------------------------------------- |
  | `\dfrac{}{}`                          | 分数命令   | `\frac{}{}`                                        |
  | `\cfrac{}{}`                          | 连分数命令 | `\frac{}{}`                                        |
  | `\begin{align}` / `\end{align}`       | 对齐环境   | 拆分为多个 `$$...$$` 块公式                        |
  | `\begin{equation}` / `\end{equation}` | 公式环境   | 用 `@formula` + `$$...$$`                          |
  | `\begin{cases}`                       | 分段函数   | 可以用 `\begin{cases}`，但需确认 Mathtype 版本支持 |

- ❌ **禁止在百分号前加转义符** — 直接写 `%`，不要写 `\%`
- ❌ **禁止在 `$$...$$` 中使用 `\displaystyle`** — 块公式默认就是 display style

## 交叉引用

### ✅ 正确写法

```
如 @ref{fig:架构图} 所示，结果见 @ref{tbl:对比}，目标函数见 @ref{eq:损失函数}。
```

### ❌ 禁止

- ❌ `@ref{xxx}` 中的标签与声明的标签不一致

---

## 参考文献

### 文中引用

#### ✅ 正确写法

```
相关研究 @cite{resnet2016,transformer2017} 表明……
```

#### ❌ 禁止

- ❌ `@cite{key}` 中的 key 不存在于 `.bib` 文件中

### 参考文献列表

#### ✅ 正确写法

```
@bibliography{refs.bib}
```

- 放在文档末尾
- 自动生成 GB/T 7714 格式的参考文献列表
